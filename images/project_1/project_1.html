Font Study

This project was done for my typography class sophomore year. The goal being to study a common type face and present it in a design feturing one letter repeatedly, the font name, serif or sans-serif, and the year it was made. I chose the font Georgia which is a serif font created in 1996, and I used the letter A so I could get some interesting angles and lines. I chose to use greyscale as I felt using bright colors would have been overwhelming with how many elements are already on the design.
